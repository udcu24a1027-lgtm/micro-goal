# Microgoal Backend 🚀

This is the backend server for the **Microgoal** application, part of the Naan Mudhalavan project. It handles user authentication and goal management.

## 🛠 Technologies Used
- **Node.js** & **Express.js** (Server)
- **MongoDB** & **Mongoose** (Database)
- **JWT** (Authentication)
- **Bcrypt.js** (Password Hashing)

## 📁 Project Structure
- `Model/` - Database schemas
- `Router/` - API endpoints
- `Middlewares/` - Auth & Error handling
- `validation/` - Input data checks

## 🚀 How to Run Locally
1. Clone the repo.
2. Run `npm install`.
3. Create a `.env` file with your `MONGO_URL`.
4. Run `npm start`.
